# 상식 퀴즈 게임 - PRD (Product Requirements Document)

## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| **제품명** | 상식 퀴즈 챌린지 |
| **버전** | 1.0 |
| **작성일** | 2026-03-31 |
| **플랫폼** | 웹 브라우저 (반응형) |
| **기술 스택** | HTML5, CSS3, JavaScript (Vanilla) |

### 1.1 목적
다양한 분야의 상식을 4지선다 객관식 형태로 풀어보며 지식을 테스트하는 퀴즈 게임을 제공한다.

### 1.2 대상 사용자
- 상식 퀴즈를 즐기고 싶은 일반 사용자
- 한국사·과학·지리·일반상식 지식을 테스트하고 싶은 학습자

---

## 2. 핵심 기능 요구사항

### 2.1 카테고리 시스템

| 카테고리 | 문제 수 | 설명 |
|----------|---------|------|
| 한국사 | 10문제 | 고대~현대 한국 역사 관련 문제 |
| 과학 | 10문제 | 물리, 화학, 생물, 지구과학 등 |
| 지리 | 10문제 | 한국 및 세계 지리 관련 문제 |
| 일반상식 | 10문제 | 사회, 문화, 스포츠, 예술 등 |
| **합계** | **40문제** | |

### 2.2 퀴즈 진행 규칙
- 각 문제는 **4개의 보기** 중 1개를 선택하는 객관식 형태
- 보기 선택 후 **즉시 정답/오답 피드백** 표시
  - 정답: 초록색 하이라이트 + 정답 메시지
  - 오답: 빨간색 하이라이트 + 정답 표시
- 한 번 선택하면 변경 불가
- 문제당 제한 시간 없음 (v1.0)
- 카테고리 순서: 한국사 → 과학 → 지리 → 일반상식 (순서대로 진행)

### 2.3 점수 시스템
- 정답 1문제당 **1점** (총 40점 만점)
- 카테고리별 소계 표시
- 오답은 0점 (감점 없음)

### 2.4 결과 및 순위

#### 최종 결과 화면
- 총점 (예: 32/40)
- 카테고리별 점수 (예: 한국사 8/10, 과학 9/10 ...)
- 정답률 퍼센트 표시
- 등급 부여:

| 점수 범위 | 등급 | 메시지 |
|-----------|------|--------|
| 36~40 | S | 🏆 상식 마스터! |
| 30~35 | A | 🥇 상식 전문가! |
| 24~29 | B | 🥈 상식 우등생! |
| 16~23 | C | 🥉 조금만 더 노력! |
| 0~15 | D | 📚 공부가 필요해요! |

#### 순위 기록
- 로컬 스토리지(LocalStorage) 기반 순위 저장
- 저장 항목: 닉네임, 총점, 카테고리별 점수, 플레이 날짜
- 상위 10명까지 순위표 표시
- 순위표는 메인 화면에서 확인 가능

## 8. 향후 확장 고려사항 (v2.0+)

> v1.0 범위에는 포함하지 않으나, 확장 가능성을 고려하여 기록한다.

- 문제당 제한 시간 추가 (타이머)
- 난이도 선택 (쉬움/보통/어려움)
- 카테고리 선택 플레이 (원하는 카테고리만)
- 문제 순서 랜덤 셔플
- 보기 순서 랜덤 셔플
- 문제 풀이 해설 추가
- 서버 기반 글로벌 순위표
- 카테고리 추가 (예술, 스포츠, IT 등)
