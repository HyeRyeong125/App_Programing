# PRD Step 2: AI 레시피 생성

## 1. 개요

1단계에서 인식된 식재료 목록을 기반으로, AI가 만들 수 있는 레시피를 추천하는 단계.
OpenRouter API의 `qwen/qwen3.6-plus:free` 모델을 활용하여 구체적인 레시피를 생성한다.

## 2. 목표

- 인식된 재료를 바탕으로 현실적이고 실용적인 레시피를 생성한다
- 사용자가 요리 조건(인원수, 난이도, 요리 종류 등)을 선택할 수 있다
- 여러 레시피를 추천하고, 선택한 레시피의 상세 정보를 제공한다

## 3. 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | HTML, CSS, JavaScript (Vanilla) |
| API | OpenRouter API |
| AI 모델 | `qwen/qwen3.6-plus:free` (텍스트 생성) |
| 데이터 | 1단계에서 전달받은 재료 목록 |

## 4. 기능 요구사항

### 4.1 요리 조건 설정

- 인원수 선택: 1인분 / 2인분 / 3~4인분 / 5인분 이상
- 난이도 선택: 초급 / 중급 / 고급
- 요리 종류: 한식 / 중식 / 일식 / 양식 / 기타
- 조리 시간: 15분 이내 / 30분 이내 / 1시간 이내 / 상관없음
- 추가 요청사항: 자유 텍스트 입력 (예: "매운 것 빼고", "다이어트용")

### 4.2 레시피 생성 (AI)

- 1단계 재료 목록 + 요리 조건을 조합하여 프롬프트 생성
- `qwen/qwen3.6-plus:free` 모델에 요청
- 3개의 레시피 추천을 한 번에 요청
- 스트리밍 응답으로 실시간 표시 (선택사항)

### 4.3 레시피 목록 표시

- 카드 형태로 3개 레시피 표시
- 각 카드에 표시할 정보:
  - 요리 이름
  - 소요 시간
  - 난이도
  - 간단한 설명 (1~2줄)
- 카드 클릭 시 상세 레시피 표시

### 4.4 상세 레시피 표시

- 필요한 재료 목록 (보유 재료 / 추가 필요 재료 구분)
- 단계별 조리 방법
- 조리 팁
- "다른 레시피 보기" 버튼 (목록으로 돌아가기)
- "레시피 저장" 버튼 (3단계 연동)

## 5. API 호출 명세

### 요청

```
POST https://openrouter.ai/api/v1/chat/completions
```

```json
{
  "model": "qwen/qwen3.6-plus:free",
  "messages": [
    {
      "role": "system",
      "content": "너는 전문 요리사이자 레시피 개발자야. 주어진 재료로 만들 수 있는 실용적인 레시피를 추천해줘. 반드시 아래 JSON 형식으로 응답해."
    },
    {
      "role": "user",
      "content": "보유 재료: [배추, 당근, 양파, 계란, 마늘]\n조건: 2인분, 초급, 한식, 30분 이내\n\n위 재료로 만들 수 있는 레시피 3개를 다음 JSON 형식으로 추천해줘:\n[{\"name\": \"요리명\", \"time\": \"소요시간\", \"difficulty\": \"난이도\", \"description\": \"간단 설명\", \"ingredients\": {\"have\": [\"보유재료\"], \"need\": [\"추가재료\"]}, \"steps\": [\"1단계\", \"2단계\"], \"tips\": \"조리 팁\"}]"
    }
  ]
}
```

### 응답 처리

- `choices[0].message.content`에서 JSON 배열 파싱
- `reasoning` 필드가 있을 경우 무시하고 `content`만 사용
- JSON 파싱 실패 시 마크다운 형태로 폴백 표시
- 에러 시 재시도 안내

## 9. 에러 처리

| 상황 | 처리 방법 |
|------|-----------|
| 재료가 없음 | "재료를 최소 1개 이상 선택해주세요" |
| API 429 (Rate Limit) | "잠시 후 다시 시도해주세요" + 자동 재시도 (10초 후) |
| JSON 파싱 실패 | 원본 텍스트를 마크다운으로 렌더링 |
| 응답이 비어있음 | "레시피 생성에 실패했습니다. 다시 시도해주세요" |
| 네트워크 오류 | "네트워크 연결을 확인해주세요" |
